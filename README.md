# membercat.com

A high quality website built all for Membercat Studios.

## Stack

-   Frontend
    -   React
    -   TailwindCSS
    -   Interia
-   Backend
    -   Laravel
-   Database
    -   SQLite

## Prerequisites

-   Node.js v18+
-   PHP v8.1+
-   Composer
-   Web Server (Apache, Nginx, etc.)
